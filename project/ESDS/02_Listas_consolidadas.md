## 7. Síntesis para conducir la reunión

La reunión debe centrarse en **cerrar alcance**, no en explorar todas las posibilidades. Elena ya ha expresado una necesidad bastante clara: web básica, visual, emocional, con SEO/localización y algún sistema sencillo de contacto o reserva.

La recomendación interna más sólida es proponer una **landing inicial ampliable**, con buena imagen, mensajes breves, SEO local básico, localización clara y contacto por WhatsApp. La automatización de reservas, pagos, blog, páginas individuales o multiidioma completo pueden quedar para una segunda fase si no son imprescindibles para julio.

La decisión clave es el **sistema de reserva/disponibilidad**. Ese punto cambia mucho el presupuesto y la complejidad. La segunda decisión clave es el **alcance inicial**: una página bien hecha frente a una web más amplia. La tercera es el **material disponible**: fotos, textos, dominio, accesos e información cerrada de las experiencias.

Fuentes de referencia usadas: **Audio** y **Mensajes/Capturas**.
