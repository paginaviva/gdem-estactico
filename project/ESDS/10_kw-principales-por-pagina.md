## Resumen de validación contra canibalización

| KW Principal | Asignada a | ¿Alguna otra página la usa? |
|:------------:|:----------:|:---------------------------:|
| `experiencias bienestar guadalest` | Home | No |
| `actividades guadalest` | /servicios/ | No |
| `mini retiro guadalest` | Mini Retiro | No |
| `tarde conexion guadalest` | Tarde Conexión | No |
| `yoga guadalest` | Yoga | No (el yoga en packs no usa esta KW como pilar) |
| `kayak embalse guadalest` | Kayak | No |
| `caminata consciente guadalest` | Caminata | No |
| `transfer guadalest` | Transfer Actividad | No (Transfer Privado usa KW distinta) |
| `transfer privado guadalest` | Transfer Privado | No |
| `el sonido del silencio guadalest` | Información | No |

**No se detecta canibalización.** Cada KW principal es única por página. Las keywords secundarias pueden solaparse entre páginas (ej: `retiro bienestar alicante` aparece en Mini Retiro y Home) pero eso es deseable para reforzar el cluster semántico, no es canibalización.

## Notas para el copywriter

- El **tono general** debe ser cálido, sensorial y en español de España (RAE). Evitar anglicismos innecesarios
- El **title tag** y el **H1** de cada página ya están definidos en su ficha. Deben respetarse tal cual
- Las **keywords principales** deben aparecer en: title tag (ya incluido), H1 (ya incluido), primer párrafo, y al menos una vez más en el body
- Las **keywords secundarias** deben integrarse de forma natural, sin forzar
- Los **datos clave** (precio, duración, horarios) deben ser exactos según la tabla de servicios del cliente
- Cada página debe **responder a la intención de búsqueda** indicada: si es transaccional, facilitar la reserva (CTA WhatsApp); si es comercial, convencer con los beneficios
- Incluir **preguntas frecuentes** (formato FAQ) en páginas donde aplique, especialmente para GEO (potencial AI Overview)
- **No copiar** la misma descripción de un servicio a otro. Cada página debe tener contenido único
